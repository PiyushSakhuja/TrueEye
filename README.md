# 🔐 TrustEye – AI-Powered Insider Threat Detection System

## 🚀 Overview

TrustEye is an advanced **cybersecurity solution** that uses Artificial Intelligence and Machine Learning to detect insider threats within organizations.

It focuses on identifying abnormal user behavior in real-time to prevent data breaches, making it a powerful tool in modern **cyber defense systems**.

---

## 🔑 Keywords

Cybersecurity, Insider Threat Detection, AI Security, Machine Learning, Anomaly Detection, User Behavior Analytics (UEBA), SIEM, Data Breach Prevention, Threat Intelligence, Network Security, Behavioral Analytics, SOC, Risk Detection, Zero Trust Security

---

## 🎯 Problem Statement

Traditional cybersecurity tools mainly focus on external attacks, but a significant number of threats originate from within the organization.

These insider threats may be:

* Malicious employees
* Negligent users
* Compromised accounts

There is a need for an intelligent system that ensures **proactive threat detection**.

---

## 💡 Solution

TrustEye provides:

* Continuous monitoring of user activities
* Machine learning-based anomaly detection
* Real-time alerts for suspicious behavior
* Adaptive learning for improved accuracy

---

## 🧠 Key Features

### 🔍 User Behavior Analytics (UEBA)

* Tracks and analyzes user activity patterns
* Detects anomalies using machine learning
* Builds baseline behavior profiles
* Identifies unusual login patterns and access behavior
* Flags insider threats early

---

### 🛡️ SIEM (Security Information and Event Management)

* Centralized log collection from systems and applications
* Real-time monitoring of security events
* Event correlation to detect complex threats
* Alert generation and incident tracking
* Audit logs for forensic analysis

---

### ⚡ Core System Features

* 🤖 AI-powered anomaly detection
* 🚨 Intelligent alert system
* 📊 Dashboard for monitoring (if implemented)
* 🔄 Continuous learning system
* 🔐 Supports Zero Trust Security

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/PiyushSakhuja/TrueEye.git
cd TrueEye
```

### 2️⃣ Install dependencies

```bash
npm install
npm start
cd ml
pip install -r requirements.txt
```

### 3️⃣ Run the project

```bash
python score_api.py
open localhost:3000
```
